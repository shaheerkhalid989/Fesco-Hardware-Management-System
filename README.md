# Fesco Hardware Management System

The Hardware Management System is a robust software solution developed in .NET C# to streamline and automate the management of a hardware warehouse at FESCO (Faisalabad Electric Supply Company). This system is designed to efficiently handle inventory tracking, stock management, procurement, and distribution of hardware components used in electrical maintenance and operations. It features user authentication, role-based access control, real-time stock updates, reporting, and automated alerts to ensure seamless operations.
